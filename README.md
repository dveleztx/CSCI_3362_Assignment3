# CSCI_3362_Assignment3

## Overview

This repository contains the third assignment for the CSCI 3362 Operating Systems course, focusing on scheduling algorithms. The project implements various scheduling algorithms to simulate and analyze their performance in an operating system environment.

## Features

- Implementation of multiple scheduling algorithms:
  - First-Come, First-Served (FCFS)
  - Shortest Job Next (SJN)
  - Priority Scheduling
  - Round Robin (RR)
- Simulation of scheduling processes
- Performance analysis of different algorithms

## Setup

To set up the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/CSCI_3362_Assignment3.git
   ```

2. Navigate to the project directory:
   ```bash
   cd CSCI_3362_Assignment3
   ```

3. Ensure you have a C++ compiler installed (e.g., `g++`).

4. Compile the project:
   ```bash
   g++ -o scheduler main.cpp
   ```

## Usage

Once compiled, you can run the scheduling simulation with the following command:

```bash
./scheduler
```

This will execute the program and display the results of the scheduling algorithms based on the given input data.

### Example

To simulate a scheduling scenario, provide the input data as specified in the program prompts. The program will output the scheduling order and performance metrics for each algorithm.
